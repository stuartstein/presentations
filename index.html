Create a new presentation

  1. Open this repository with your coding agent.
  2. Tell it the presentation topic, audience, source context, desired length, event/date, and constraints.
  3. Ask it to create a new deck using the existing presentation style.
  4. Review the local preview, leave comments, and ask the agent to iterate.
  5. When it looks good, ask the agent to commit, push, and verify the published page.

The agent already has internal repo instructions for creating the deck, adding it to the homepage, and preserving the shared design system.
